# Roadrunner Chassis GPIOs

| GPIO Description | GPIO Group | GPIO Num | GPIO Pin¹ |
|------------------|------------|----------|-----------|
| Reset button     | 3          | 19       | 83        |
| Green Power LED  | 2          | 29       | 61        |
| Red Power LED    | 2          | 28       | 60        |
| Blue Status LED  | 3          | 20       | 84        |
| Red Status LED   | 3          | 11       | 75        |
| Blue Link LED    | 3          | 14       | 78        |
| Red Link LED     | 3          | 10       | 74        |

¹ Pin = ( Group - 1 ) * 32 + Num
